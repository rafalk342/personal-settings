# Operating System

cmd + space -> Show system search

cmd + tab -> switch window

# Web Browser

j - scroll down

k - scroll up

u or caps + i - page up

d or caps + u - page down

caps + o - home

caps + p - end

H - go back in history

L - go forward in history

J - go one tab left

K - go one tab right

# Obsidian

cmd + e -> toggle reading view

# Terminal
cmd + 1 -> switch to first tab
